#  fast-exp

An implementation of fast exponentiation that can calculate exponential for big numbers in linear time complexity
## Install

```
npm install fast-exp
```

## Usage

```
const fastExp = require("fast-exp")
console.log(fastExp(2,10)) //1024
```
